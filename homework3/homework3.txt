Introduces UNIX tools to both experienced computer users and to students with only a basic knowledge of computers. This course provides orientation to the UNIX operating system; the UNIX file system; electronic mail; standard UNIX editors; text and information processing; file and directory organization, management and manipulation; and standard UNIX utilities. Students write simple UNIX shell programs (scripts). This couse also covers advanced forms of commands and utilities, which include, sed, preg, awk, as well as new commands and constructs. Students learn regular expressions as well as constructs and special commands used in writing shell scripts. Topics include functions, traps, arithmetic on variables, and input/output techniques. Emphasis is on understanding how the UNIX shell operates. Time permitting, PERL is studied.
